## 发布命令与判定

本地发布门禁：

```bash
npm run verify:local
```

只有以下远程步骤全部成功后，才能把“真实在线系统”标为通过：

```bash
npm run supabase:push
npm run supabase:sync-plan
npm run supabase:verify
```

随后还需验证部署域名上的手机/桌面登录、Realtime 刷新和四账号完整闭环。远程门禁使用合成账号，不导入真实孩子隐私数据。
